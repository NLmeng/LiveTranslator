# LiveTranslator
Translate texts and overlay on screen.

### Getting Started

#### Environments
- Install Dependencies: `pipenv install`
- Use virtual environment: `pipenv shell`
- Exit/Deactivate: `exit`

#### Start
- `python3 main.py`


#### Troubleshoot
- `pipenv --rm` + `pipenv --python 3.9` + reinstall dependencies
